# crossover-stockton
A static, open-source infrastructure project designed to map the reality of Stockton's support services. Built on a GitOps data pipeline, it provides a fast, anonymous, legible dashboard of local resources for both human outreach workers and AI agents.

## Quick Start
To start the local development server without caching or URL redirects:
'npx http-server . -p 3000 -c-1'
